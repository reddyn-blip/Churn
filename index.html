The scoring model

FLAG if shipments drop >30% below seller's OWN 2-month rolling norm
OR RTO rate >25% OR current shipments ≤ 1
Only sellers with history in at least one prior month are scored.

Baseline = rolling average of forward shipments across the 2 most recent non-current months in the ZIP.

Current period = the file whose month/year is closest to today.

RTO rate = rows where shipment_type = RTO ÷ total shipments for that seller in the current period.

Score is additive: drop severity (0–50) + RTO severity (0–35) + minimal activity flag (20). Tier: High ≥ 60, Medium ≥ 25, Low < 25.
